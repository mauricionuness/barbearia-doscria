
Instruções para publicar no GitHub Pages (grátis)

1) Crie um repositório no GitHub (público ou privado).
2) No seu computador, descompacte este arquivo .zip e coloque todo o conteúdo na pasta do repositório.
3) Faça commit e envie (push) para o GitHub.
4) No GitHub, vá em Settings → Pages → Branch: selecione 'main' (ou 'master') e / (root).
5) Clique em Save. Em alguns minutos o site estará disponível em: https://<seu-usuário>.github.io/<seu-repo>/

Observações:
- O painel administrativo usa localStorage do navegador para salvar barbeiros e agendamentos.
- Senha padrão do admin: admin2025 (você pode alterar no arquivo admin.html se quiser outra).
- Para backups, use Exportar no painel admin e salve o JSON. Para restaurar, use Importar.
- Google Maps é embutido via iframe, sem necessidade de API Key.
- Para alterar textos, preços ou imagens, você pode editar os arquivos .html / .css diretamente e dar push novamente.
